## 요약:

- VPC 엔드포인트를 통해, VPC와 엔드포인트 지원 서비스 간에 연결 설정 가능 (IGW, NAT, VPN, DX 전부 불필요)
  - VPC에 연결할 수 있는 특정 API 엔드포인트, 사이트 및 서비스 제어 가능

<br>

- VPC 엔드포인트는 가상 디바이스
  - 수평으로 확장되는 고가용석 중복 VPC 구성 요소
